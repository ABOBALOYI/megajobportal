B) LESS/CSS Files and Structure

I'm using Four CSS and One LESS file in this template. Those are as follows

  • bootstrap.min.css
  • font-awesome.min.css
  • slick.css
  • slick-theme.css
  • app.less

bootstrap.min.css file is default bootstrap css file it contains all default classes from bootstrap.

font-awesome.min.css file is used for including the fontawesome icons.

slick.css and slick-theme.css file are used for animating client logos list at banner bottom and alpha filters button on company listing page.

app.less file has all the supporting less files included in it.

layout.less file contains majority of code and is very well documented. you will find all section CSS in this file only. This file contains global and section wise css styles

responsive.less file is used to make the page responsive across all devices.

If you would like to edit the colors of the template you should go to variables.less to make the color changes

If you would like to edit the font or style of any elements, you should go to layout.less to make the desired changes

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Note All the images included into the page are not for sell. you need to replace these images with your own images inside images folder.

C) JavaScript

This template imports following Javascript files.

  • jquery.min.js
  • bootstrap.min.js
  • slick.min.js
  • jquery.countdown.min.js
  • summernote.min.js
  • custom.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. bootstrap.min.js file is a library Javascript file provided by Bootstrap and must be included to make use of bootstap plugins
  3. slick.min.js file is included to make the clients icons slider at banner bottom and alpha filter buttons on the company lisiting page.
  4. jquery.countdown.min.js is included for the countdown timer on comming soon page.
  5. summernote.min.js is included for text editor on the create resume page.
  6. custom.js file is used to initialize different functions.
